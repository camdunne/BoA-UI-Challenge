## Running Application
Keep open in terminal:
### yarn
```sh
yarn start:dev
```

### npm
to run bundle:
```sh
npm run compile
```
to run server:
```sh
npm run server
```

### test
testing with jest, to run:
  with yarn:
```sh
yarn test
```
  or with npm:
```sh
npm test
```
